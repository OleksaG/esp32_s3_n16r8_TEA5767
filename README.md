Можливості коротко: 
- обирати чутливість сканування і автоматично сканувати ФМ діапазон з 87.5 МГц
- зберігати на швидкі кнопки 21 хвилю і підписувати їх кирилицею
- повторно сканувати діапазон без втрати підписаних станцій
- піднастроювати хвилі +/-0,1Мгц
- додавати і видаляти станції у ручному режимі
- налаштування не залежать від живлення

Докладніше у pdf файлі.
